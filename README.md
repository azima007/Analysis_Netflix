# Analysis_Netflix


Проект по анализу данных

Проект: Анализ контента Netflix
Автор: Жангильхан Азима

1. Резюме проекта

Данный проект посвящён анализу данных о фильмах и сериалах Netflix.
Цель — выявить ключевые закономерности, связанные с типами контента, странами-производителями, жанрами, возрастными рейтингами и динамикой релизов.
Проект выполнялся для экзамена в AlmaU.

2. Бизнес-вопросы, на которые мы отвечаем

#1. Какой тип контента доминирует на Netflix: фильмы или сериалы?
#2. Какие страны производят наибольшее количество контента?
#3. Какие жанры (listed_in) являются самыми популярными?
#4. Как распределяется контент по возрастным рейтингам (rating)?
#5. Как изменилась динамика релизов контента по годам?
#6. Какие режиссёры представлены чаще всего?
#7. Какие страны чаще других выпускают сериалы, а какие — фильмы?
#8. Какие категории жанров наиболее востребованы среди пользователей?
#9. Есть ли различия в распространённости контента между разными странами?
#10. Какие направления контента (Documentaries, Dramas, Comedies и т.п.) формируют ядро библиотеки Netflix?
